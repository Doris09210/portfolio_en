Research on the Multi-Chart Layout Design of Large-Screen Interface

HCII 2025 · First Author

Accepted

This study addresses the multi-chart layout design problem for large-screen interfaces, using Unity to build interactive experimental environments and Linear Mixed Models (LMM) for cognitive load assessment. The research proposes multi-chart layout optimization principles based on cognitive load theory, providing scientific basis for complex information interface design.

Research Methods

Unity Experiment LMM Analysis NASA-TLX

Key Findings

  • • Optimized layouts can reduce cognitive load
  • • Information density and cognitive load show nonlinear relationship
  • • Visual hierarchy structure affects information processing efficiency

Designing RLHF Interfaces for Efficient and High-Quality Human Feedback

Submitted · First Author

Submitted

This study focuses on interface design issues in Human Feedback Reinforcement Learning (RLHF), constructing conversational RLHF interfaces based on Shared Mental Models (SMM) theory. By integrating behavioral data and eye-tracking data, deeply explores the influence mechanism of feedback transparency on cognitive load.

Theoretical Framework

Shared Mental Models RLHF Cognitive Load Conversational UI

Expected Contributions

  • • Construct SMM-driven RLHF interactive cognitive framework
  • • Design verifiable conversational UI solutions
  • • Vertically trained models
  • • Design strategies to improve human-computer collaboration efficiency
